# Recommender Systems

Projects implementing recommendation algorithms and collaborative filtering techniques.

## Projects

### Book Recommendation System
Content-based and collaborative filtering system for book recommendations.

**Techniques:**
- Content-based filtering
- Collaborative filtering
- Similarity metrics (cosine similarity, Euclidean distance)
- Matrix factorization

**Dataset:** Book ratings and metadata

## Skills Demonstrated

- Rating predictions
