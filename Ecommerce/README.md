# Olist Brazilian E-Commerce Data Analysis

## Overview
This project delivers a full-stack exploratory data analysis and business audit of the Olist Brazilian E-Commerce dataset. The notebook covers:
- Data engineering (merging, cleaning, feature engineering)
- Logistics and operations analysis
- Customer segmentation (RFM, cohort analysis)
- NLP on customer reviews
- Sales, seasonality, and Pareto analysis
- Predictive modeling (Random Forest)
- Actionable business recommendations

## Key Insights
- Regional delivery delays and logistics bottlenecks
- Customer retention and segmentation
- Root causes of negative reviews (delivery, product quality)
- Sales concentration among top sellers
- Predictors of late delivery

## How to Use
1. Open `Olist_Ecommerce_Analysis.ipynb` in Jupyter or VS Code.
2. Run all cells from top to bottom for a complete analysis.
3. Review markdown commentary for business insights and recommendations.

## Requirements
- Python 3.7+
- pandas, numpy, matplotlib, seaborn, scikit-learn, wordcloud

## File Structure
- `Olist_Ecommerce_Analysis.ipynb`: Main analysis notebook
- Olist CSV files: All required datasets (see notebook for filenames)
