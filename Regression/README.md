# Regression Projects

Machine learning projects focused on predicting continuous variables using various regression techniques.

## Projects

### Finance (Credit Risk Analysis)
Analyzing credit risk factors and predicting loan defaults.

**Techniques:** Logistic Regression, Decision Trees
**Dataset:** Credit risk dataset with customer financial data

### Loan Approval System
Automated loan approval prediction based on applicant characteristics.

**Techniques:** Classification models, Feature engineering
**Dataset:** Loan application data

### Diabetes Prediction
Predicting diabetes occurrence based on medical indicators.

**Techniques:** Classification/Regression models
**Dataset:** Medical diagnostic measurements
