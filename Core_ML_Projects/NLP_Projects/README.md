# Natural Language Processing Projects

NLP and text analytics projects demonstrating various text processing and analysis techniques.

## Projects

### Resume Screening NLP
Automated resume screening using NLP techniques to classify and match candidates.

**Techniques:** Text classification, TF-IDF, vectorization

### SMS Spam Detection
Binary classification to identify spam messages.

**Techniques:** Naive Bayes, text preprocessing, feature extraction

### Language Classification
Multi-class classification to identify the language of text.

**Techniques:** Character n-grams, language models

### Text Summarization
Automatic text summarization using extractive methods.

**Techniques:** Text ranking, sentence scoring

### US Election Sentiment Analysis
Sentiment analysis on election-related tweets.

**Techniques:** Sentiment classification, social media analysis, data visualization

### WhatsApp Sentiment Analysis
Analyzing sentiment in chat conversations.

**Techniques:** Chat parsing, sentiment detection, temporal analysis

## Common NLP Skills
- Text preprocessing (tokenization, stemming, lemmatization)
- Stop word removal
- TF-IDF vectorization
- Word embeddings
- Sentiment analysis
- Classification models
- Regular expressions
- Language detection
